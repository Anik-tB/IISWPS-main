# Frontend - Industrial Safety System UI

React + TypeScript frontend for the Intelligent Industrial Safety & Workflow Prediction System.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Features

- Real-time dashboard with sensor monitoring
- Interactive route planner with A* visualization
- Machine parameter optimizer
- Risk classifier interface
- Boolean logic simplifier

## Technology Stack

- React 18
- TypeScript
- Vite
- TailwindCSS
- Zustand
- Recharts
- Leaflet
- React Router

## Project Structure

- `src/pages/` - Page components
- `src/components/` - Reusable components
- `src/api/` - API client functions
- `src/store/` - State management
